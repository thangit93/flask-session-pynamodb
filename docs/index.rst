Flask-Session
=============

Flask-Session is an extension for `Flask`_ that adds support for server-side sessions to
your application.

.. _Flask: http://flask.palletsprojects.com/


Installation
------------

Install from PyPI using an installer such as pip:

.. code-block:: text

    $ pip install Flask-Session
